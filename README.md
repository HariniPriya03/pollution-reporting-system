# 🌍 Pollution Reporting System

A full-stack web application that enables users to report pollution incidents in real time, helping authorities and communities respond efficiently.

---

## 🚀 Features

* 📝 Report pollution incidents with detailed information
* 🔐 User authentication (Login system)
* 📊 Dashboard to view and manage reports
* 🏠 Home page with overview of the system
* ⚡ Interactive UI built with React

---

## 🛠️ Tech Stack

**Frontend:**

* React.js
* JavaScript, HTML, CSS

**Backend:**

* Node.js, Express.js

**Database:**

* MongoDB

**Tools:**

* Git, GitHub

## ⚙️ Installation & Setup

### 1. Clone the repository

```bash id="zlg2hx"
git clone https://github.com/HariniPriya03/pollution-reporting-system.git
cd pollution-reporting-system
```

---

### 2. Setup Frontend

```bash id="cizuf5"
cd frontend
npm install
npm start
```

---

### 3. Setup Backend

```bash id="7shnjg"
cd backend
npm install
node server.js
```
